# Stremio for LG webOS

Unofficial webOS package of Stremio.

## Features

- Runs on LG webOS
- Homebrew compatible
- Internet permission enabled

## Build

```bash
git clone https://github.com/TGiaDo/stremio-webos.git
cd stremio-webos
npm install
npm run build
./scripts/package.sh
```

## Install

Install the generated `.ipk` using Homebrew Channel.

## Tested on

- LG webOS 4.0
